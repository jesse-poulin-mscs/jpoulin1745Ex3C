Build a PC

The first step towards building a pc is to select what parts you want to build it with.
PCPartPicker is a great place to start/plan out a system build.
The bare minimum each PC needs in order to function is to have Motherboard, RAM, CPU, CPU cooler, GPU, PSU, case, and a hard drive or ssd. To read about the definitions and functions of each of those go to Definitions.
Once you understand the terms and functions you'll need to start obtaining everything you need for a PC.
In general, to have a fully functional PC, you'll need a GPU, CPU, PSU, some RAM, storage, a case and a motherboard.
